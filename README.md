## 1. Обзор проекта

Puzzle Sokoban 3D — это минималистичный 3D-паззл с классической механикой толкания ящиков. Цель — продемонстрировать чистую архитектуру Unity-проекта: паттерн Command с Undo/Redo, MVC, UniTask для асинхронных анимаций и ScriptableObject-driven дизайн уровней.
